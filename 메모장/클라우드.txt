- VM을 생성하면 많은 부가 서비스들이 생성

Cloud에서 VM설치후 xrdp를 통해서 접속 

sudo yum -y install dnf 		         <== dnf 설치 
sudo yum -y groupinstall "Server with GUI"      <== Linux GUI 설치 

xrdp 	<== Linux를 windows 원격 데스크탑으로 접속해서 GUI 

방화벽 설정 < == TCP 3389 포트 오픈 

-xrdp 설치 
sudo dnf install epel-release		<== xrdp를 설치하기 위한 레파지토리 변경
sudo dnf install xrdp 		<== xrdp 설치

--리부팅시 서비스 자동 시
sudo systemctl enable xrdp --now
sudo systemctl restart xrdp		<== xrdp 서비스 시작
